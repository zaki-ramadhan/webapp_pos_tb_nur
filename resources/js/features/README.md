# Frontend Feature Structure

Folder `resources/js/features` dipakai untuk komponen dan logic frontend per modul bisnis:

- `catalog`
- `auth`
- `inventory`
- `purchasing`
- `sales`
- `finance`
- `shared`

Folder `resources/js/components/ui` dipakai untuk UI primitives lintas fitur, seperti button, input, checkbox, divider, panel, section heading, modal base, tooltip, dropdown menu, card, loading state, empty state, spinner, dan notice.

Page Inertia tetap berada di `resources/js/pages`, sedangkan layout umum berada di `resources/js/layouts`.
