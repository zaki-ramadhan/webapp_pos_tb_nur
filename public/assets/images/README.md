# Image Assets

Struktur folder image untuk aset UI proyek ini:

- `auth/carousel`
  Untuk gambar slider panel kiri pada halaman login atau daftar.
- `auth/branding`
  Untuk logo, mark, atau aset visual khusus area auth.
- `shared`
  Untuk image umum yang dipakai lintas halaman.

Contoh path yang nanti bisa langsung dipakai:

- `/assets/images/auth/carousel/slide-01.jpg`
- `/assets/images/auth/branding/logo.png`
- `/assets/images/shared/app-background.jpg`
- `/assets/images/shared/empty-state.png`
